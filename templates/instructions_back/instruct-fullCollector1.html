Collector Instructions

Welcome to the Collector Game!

Each time you give a correct response, you will collect some number of gems. The number of gems added for each correct response will differ across turns.

On the LEFT side of the bank, a symbol next to the GREEN CHECK MARK will show you how many gems you can add to your bank for each correct response.

On LOW-VALUE turns, each correct response adds gem.



On HIGH-Value turns, each correct response adds gems.
